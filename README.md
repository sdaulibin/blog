# 📝 Blog

> 个人博客文章仓库，使用 Obsidian 写作，GitHub 托管

## ✨ 关于

这是我的个人博客文章仓库，记录日常思考、技术探索和生活感悟。

**工作流程**：Obsidian 本地写作 → iCloud 同步 → GitHub 托管分享

## 📂 文章结构

```
blog/
├── 2026/               # 按年份归档
│   ├── 文章1.md
│   └── ...
├── assets/             # 图片等静态资源
└── .obsidian/          # Obsidian 配置（已忽略敏感配置）
```

## 🛠️ 本地环境

**推荐编辑器**：[Obsidian](https://obsidian.md/)

1. 克隆仓库
   ```bash
   git clone https://github.com/sdaulibin/blog.git
   ```
2. 用 Obsidian 打开文件夹作为 Vault
3. 开始写作！
